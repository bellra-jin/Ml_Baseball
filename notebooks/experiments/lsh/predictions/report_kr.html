2. TPOT 자동 머신러닝 파이프라인

TPOT(Tree-based Pipeline Optimization Tool)은 유전 알고리즘을 사용하여 최적의 머신러닝 파이프라인을 자동으로 탐색합니다. 데이터에 최적화된 아키텍처를 찾기 위해 전처리, 특성 공학 및 모델 알고리즘의 수천 가지 조합을 테스트합니다.

2.1 최적화된 파이프라인 단계

3세대 진화(개체 수=20)를 거쳐 TPOT은 다음과 같은 파이프라인으로 수렴되었습니다:

[단계 1] 정규화 (Normalization) Normalizer - L2 정규화(행 기준)를 통해 특성들을 비교 가능한 스케일로 조정
[단계 2] 특성 선택 (Feature Selection) SelectPercentile(87%) - 가장 정보력이 높은 상위 87% 특성 유지 (36개 중 31개)
[단계 3] 특성 결합 (Feature Union) FeatureUnion - 기본 통과 데이터와 함께 ZeroCount 특성 추가
[단계 4] 특성 결합 (Feature Union) FeatureUnion - SkipTransformer와 기본 통과 데이터 조합
[단계 5] 분류기 (Classifier) LogisticRegression - L1 규제(LASSO), 클래스 가중치 균형(balanced), SAGA 솔버 사용

2.5 모델 성능 평가

본 모델은 분류 모델(이진: 포스트시즌 진출 여부)이므로 R-squared와 같은 회귀 지표 대신 분류 지표를 사용하여 평가합니다.

2.5.1 ROC 곡선 (Receiver Operating Characteristic)

ROC 곡선은 모든 가능한 임계값에서 진짜 양성 비율(진출 적중)과 가짜 양성 비율(오탐지) 사이의 트레이드오프를 보여줍니다. AUC(곡선 아래 면적)가 1.0에 가까울수록 완벽한 분리 성능을 의미합니다.

2.5.2 학습 곡선: 훈련 데이터 크기에 따른 로그 손실

학습 곡선은 훈련 데이터의 양에 따른 로그 손실(예측 오차)의 변화를 나타냅니다. 학습 곡선과 검증 곡선 사이의 간격이 좁아지는 것은 좋은 일반화 성능을 의미합니다.

학습 로그 손실(파란색)은 데이터가 많아질수록 꾸준히 감소합니다. 학습과 검증 사이의 간격이 좁아지며, 모델이 추가 데이터로부터 혜택을 받고 있음을 보여줍니다.

학습 곡선 주요 발견:

  • 단조적 개선: 훈련 데이터가 10%에서 100%로 증가함에 따라 학습 및 검증 로그 손실이 모두 감소합니다. 더 많은 시즌 데이터가 더 나은 예측으로 이어집니다.
  • 일반화 간격 (최종): 0.379 - 단 8개 시즌의 시계열 데이터임을 감안할 때 이 간격은 건강한 수준이며, 심각한 과적합이 없음을 시사합니다.
  • 수익 체감 지점: 약 60% 이상의 데이터(5개 시즌 이상)가 확보된 시점부터 검증 손실 개선이 완만해집니다. 이는 KBO 데이터를 활용한 예측 시 최소 5~6개 시즌이 필수적임을 나타냅니다.

2.5.3 정밀도-재현율 곡선 (Precision-Recall Curve)

정밀도(진출 예측 중 실제 진출 비율)와 재현율(실제 진출 팀 중 예측 성공 비율)의 균형을 평가합니다.

학습 AP = 0.927 | 검증 AP = 0.724. 회색 점선은 베이스라인(10개 팀 중 5개 진출 = 0.50)입니다.

2.5.4 캘리브레이션 곡선 (신뢰도 도표)

예측된 확률이 실제 결과와 얼마나 잘 일치하는지 측정합니다. 완벽한 모델은 대각선 점선을 따릅니다. 예를 들어, 70% 확률로 예측된 팀들은 실제로 약 70%의 빈도로 진출해야 합니다.

학습 단계의 캘리브레이션은 거의 완벽합니다. 검증 단계에서는 약간의 과신(예측 확률이 실제 빈도보다 높음) 경향이 관찰됩니다.

해석: 검증 캘리브레이션 곡선이 대각선 아래에 위치하므로, 모델이 2025년에 대해 약간 과신하는 경향이 있습니다. 브라이어 스코어 0.260(기준점 0.25 근처)은 이것이 허용 범위 내에 있음을 확인해 줍니다. 2026년 예측치를 해석할 때 "LG 96.9%"와 같은 확률은 다소 낙관적일 수 있음을 인지해야 합니다.

dyn_ 변수의 혁신: dyn_k = (1 - 경기 소화율) x 3년 평균_k 공식을 사용합니다. 시즌 초반(소화율 ~ 0)에는 3개년 기록이 온전히 반영되지만, 시즌 종료 시점(소화율 ~ 1)에는 가중치가 0으로 수렴합니다. 이는 경기 데이터가 쌓임에 따라 예측의 무게중심을 과거의 전력에서 현재의 성적으로 우아하게 전환합니다.

4. 모델 상세: L1 규제 로지스틱 회귀

유전 알고리즘을 통해 3세대 동안 60개의 후보 파이프라인을 탐색한 결과, L1(LASSO) 규제가 적용된 로지스틱 회귀가 최적 모델로 선정되었습니다.

알고리즘: LogisticRegression(C=1330, penalty=L1, solver=saga, class_weight=balanced, max_iter=1000)
L1 규제를 사용하는 이유: LASSO 규제는 중요하지 않은 특성의 계수를 0으로 만들어 자동 특성 선택을 수행하므로, 모델이 희소(sparse)해지고 해석 가능성이 높아집니다.
Balanced 클래스 가중치: 팀별 라벨 불균형(50:50이 이상적이나 미세한 차이 존재)을 보정합니다.
학습 성능: AUC = 0.9234 | 정확도 = 83.55%

4.1 L1 규제 설명 (비기술적 관점)

L1 규제(LASSO)는 36개의 특성 중 실제로 중요한 것이 무엇인지 식별합니다. 모든 특성을 사용하는 대신 약한 특성을 선별적으로 제거합니다. 이는 마치 야구 감독이 선발 라인업을 짜는 것과 같습니다. 36개의 모든 통계가 똑같이 중요하지는 않기 때문입니다. 모델은 5~7개의 핵심 지표(5위와의 승차, 득실차 이력 등)가 대부분의 예측력을 가지며, 10경기 반짝 승률과 같은 단기 변동성은 덜 중요하다는 것을 스스로 학습합니다.

4.2 TimeSeriesSplit 검증

시계열 데이터에서 표준 K-Fold 교차 검증을 사용하면 미래 정보가 과거로 유출될 수 있습니다. 대신 TimeSeriesSplit을 사용하여 각 "폴드"가 과거 데이터만을 사용하도록 보장했습니다:

  • Fold 1: 2017~2019 학습 -> 2020 예측
  • Fold 2: 2017~2020 학습 -> 2021 예측
  • Fold 3: 2017~2021 학습 -> 2022 예측
  • Fold 4: 2017~2022 학습 -> 2023 예측
  • Fold 5: 2017~2023 학습 -> 2024 예측
  • 최종 검증 (2025): 2017~2024 학습 -> 2025 예측 (마일스톤 평가에 활용)

이러한 점진적 설계는 시즌을 거듭하며 변화하는 리그 차원의 ERA/OPS 트렌드를 자연스럽게 포착합니다.

5. 특성 중요도 - SHAP 분석

SHAP(SHapley Additive exPlanations) 값은 각 특성이 개별 예측에 기여하는 정도를 수치화합니다. 평균 |SHAP| 값이 높을수록 모델의 포스트시즌 진출 결정에 더 큰 영향을 미칩니다.

평균 |SHAP| 값 기준 상위 15개 특성. 당해 시즌 순위 및 승차 지표가 주도하며, 3개년 및 직전 시즌 득실차가 그 뒤를 잇습니다.

가장 중요한 5대 특성

순위특성그룹해석
1games_behind_5th 당해 시즌 5위 팀과의 승차. 포스트시즌 진출의 가장 직접적인 척도: 음수는 5위 이내, 양수는 5위 밖을 의미. SHAP 값 0.108.
2dyn_run_differential 3개년 동적 시즌 진행에 따른 가중치 감쇄가 적용된 3개년 평균 득실차. 팀의 지속적인 전력을 포착. SHAP 값 0.078.
3prev_run_differential 직전 시즌 직전 시즌의 득실차. 과거 PS 진출팀 평균(+45.7)과 비진출팀(-41.9) 사이의 큰 격차(87.6) 반영. SHAP 값 0.074.
4games_behind 당해 시즌 1위와의 승차. 현재 시즌 내에서 팀의 전반적인 전력 수준을 나타내는 광범위한 지표. SHAP 값 0.056.
5wins_to_5th 당해 시즌 5위 탈환을 위해 필요한 승수. games_behind_5th를 보완하는 지표. SHAP 값 0.041.

SHAP 분석 주요 결과

  • 순위 격차 지표의 압도적 영향: 상위 5개 특성이 모두 순위 또는 승차와 관련되어 있습니다. 이 지표들이 전체 SHAP 중요도의 72%를 차지합니다.
  • dyn_run_differential (#2)의 높은 순위는 dyn_ 설계의 유효성을 증명합니다. 모델은 시즌이 진행됨에 따라 자연스럽게 소멸하는 다년 성적의 지속성 신호에 의존합니다.
  • prev_run_differential (#3)은 EDA 결과와 일치합니다: 전년도 득실 마진은 당해 시즌 순위를 고려한 후에도 강력한 예측 인자로 작용합니다(r=+0.402).
  • 흥미로운 점: 단순 승률(win_rate, SHAP = 0.0001)은 거의 영향을 주지 못합니다. LASSO 규제가 games_behind_5th나 순위 지표와 중복되는 정보를 담고 있는 승률의 계수를 0으로 수렴시켰기 때문입니다.

6. 2025 시즌 참조 차트

2025년 전체 시즌 승률 궤적. [PS]는 실제 포스트시즌 진출팀(굵은 선), 나머지는 점선으로 표시.

차트는 모델의 논리를 뒷받침합니다: 일찍 치고 나가는 팀(0.550 이상을 유지한 LG, 상승세를 탄 한화)은 상위권에 안착하는 반면, 0.450 아래로 떨어진 팀들은 지속적으로 탈락권에 머뭅니다. 2025년 PS 진출군(LG 0.603, 한화 0.593, SSG 0.536, 삼성 0.521, NC 0.514)은 모두 0.500 이상의 승률을 유지했습니다.

M4(144경기) 시점에서 모델은 90%의 정확도(10개 팀 중 9개 적중)와 AUC 1.0(완벽한 순위 분리)을 달성했습니다. 유일한 오답은 5위권에서 아주 근소한 차이로 탈락한 팀이었습니다.

7. 주요 시사점 및 향후 과제

주요 발견

  • 해석 가능성의 승리: TPOT은 복잡한 앙상블 모델 대신 단순하고 해석 가능한 로지스틱 회귀를 선택했습니다. 모델은 어떤 통계(5위와의 승차, 역사적 득실차 등)가 진출을 결정짓는지 투명하게 보여줍니다.
  • dyn_ 변수의 검증: dyn_run_differential이 SHAP 중요도 2위를 차지하며, 3개년 평균의 점진적 감쇄 방식이 과거 전력과 현재 폼의 균형을 잘 잡고 있음을 확인했습니다.
  • 역사적 전력의 중요성: prev_run_differential(#3) 지표는 현재 순위를 반영한 후에도 전년도 팀 전력이 상당한 예측 가중치를 가짐을 시사합니다.
  • 단조 학습 곡선: 마일스톤 평가(M1=0.50 -> M4=0.90)를 통해 시즌이 진행됨에 따라 모델의 확신도가 적절히 상승함을 입증했습니다.

한계점 및 향후 계획

  • 시즌 초반 예측의 노이즈: 2026년 예측은 현재 24~25경기에 기반하고 있습니다. M1 단계(36경기 시점)의 정확도가 0.50인 점을 고려할 때, 현재 수치는 불확실성이 큽니다. M2(72경기) 및 M3(108경기) 마일스톤에서 재평가가 필요합니다.
